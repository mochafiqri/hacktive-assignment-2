CRUD ORDER

## API Reference

#### GET order

```http
  GET /order/{order_id}
```

#### POST order

```http
  POST /order
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `data`    | `json` | **Required**. Id of item to fetch |


#### PUT order

```http
  PUT /order
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `data`    | `json` | **Required**. Id of item to fetch |


#### DELETE order

```http
  DELETE /order/{order_id}
```
